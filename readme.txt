Integrazione pagamenti Woocommerce WebcomPro

== Description ==

Plugin wordpress per pagare con webcompro nel proprio negozio woocommerce.


== Installation ==

1. Prendi nota del token di accesso alle api Webcom Pro
2. Installa il plugin woocommerce tramite lo store dei plugin WordPress.
3. Carica i file del plugin nella cartella `/wp-content/plugins/webcompro`, oppure installa il plugin tramite lo store dei plugin WordPress.
4. Dalla pagina `Plugins` attiva il plugin, Ã¨ consigliato abilitare gli aggiornamenti automatici
5. Attiva il metodo di pagamento Webcom Pro dalla pagina della configurazione dei pagamenti di woocommerce ed inserisci il tuo token nella configurazione
